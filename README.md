# -Personal-Finance-Manager

A command-line based **Personal Finance Manager** built using **Python**, designed to help users track daily expenses, analyze spending patterns, and manage financial data efficiently using CSV file persistence.

---

##  Project Overview

Managing personal expenses manually can be difficult and error-prone.  
This project provides a **simple yet powerful command-line application** that allows users to:

- Add and store expenses
- View all expenses
- Generate category-wise and monthly reports
- Search expenses using keywords
- Backup expense data safely

The application follows **modular programming principles** and uses **CSV files** for data persistence.

---

##  Technologies Used

- **Python 3**
- **CSV Module** (file handling & persistence)
- **Command Line Interface (CLI)**
- **Object-Oriented Programming (OOP)**
